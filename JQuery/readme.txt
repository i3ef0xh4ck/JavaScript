JQuery研究


环境搭建：
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <script type="text/javascript" src="./node_modules/jquery/dist/jquery.js"></script>
    <title>Title</title>
</head>
<body>
    <script type="text/javascript">alert($)</script>
</body>
</html>

如果使用该页面打开浏览器，弹出以下 信息，说明已经搭建好了
